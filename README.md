# demo-comark-tiptap

Small demo showcasing a potential [`@comark/tiptap`](./lib/comark-tiptap) and [`@comark/tiptap-vue`](./lib/comark-tiptap-vue) integration.

This demo also wraps both into a Nuxt UI compatible playground.

You can find the main discussion at [`comarkdown/comark#164`](https://github.com/comarkdown/comark/issues/164).
